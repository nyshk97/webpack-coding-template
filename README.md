## Required
- npm

## Get Started
```
$ git clone https://github.com/nyshk97/webpack-coding-template.git
$ mv webpack-coding-template project-name
$ cd project-name
$ npm i
```

## Start development server
```
$ npm start
```

## Build
Output with basic style
```
$ npm build:dev
```
or Output with one line
```
$ npm build
```
